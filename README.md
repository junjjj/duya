# DuYa
搭建一个平台用于学习 `DDD` 架构模式和 `monorope` 项目结构，实现插件化方案（其他方向）

实现一个功能，就是在网页能够创建项目、构建项目以及发布项目

实现的技术：Vue.js 3.x + Vue-router + Vuex
